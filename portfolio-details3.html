Sudoku Puzzle Solver (Java)

View on GitHub

A classic Sudoku puzzle solver developed in Java using recursive backtracking and constraint satisfaction techniques to find complete and valid solutions for any partially filled 9x9 grid.

The application reads a 9x9 Sudoku puzzle and identifies unassigned cells. It then uses a recursive depth-first search algorithm to test number placements, backtracking when constraints are violated to ultimately solve the puzzle.

The key challenge was ensuring the algorithm efficiently handled edge cases and invalid inputs while maintaining optimal performance even in recursive loops.

Implemented a depth-first recursive backtracking solution in Java that checks constraints for each placement, offering a minimal yet effective approach to Sudoku solving. The code ensures a fully filled and valid board at the end.

Key Features

  • 9x9 Puzzle Solver
  • Recursive Backtracking
  • Constraint Checking
  • Pure Java Implementation
  • Terminal-Based Output
  • Fast Execution
